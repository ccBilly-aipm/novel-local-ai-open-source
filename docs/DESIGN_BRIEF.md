# Novel Local AI — 视觉改版设计简报（给 Claude Design 用）

> 目标风格：**赛博朋克 2077 / 《边缘行者》(Edgerunners) 动画**那种霓虹、暗黑、高对比、炫酷的科幻感。
> 当前风格：暖纸质 / 编辑部气质（要被替换掉）。
> 技术栈：React + Vite + Tailwind v3.4，桌面优先（min-width 1024），目前**无暗色模式**（改版本身就是引入暗色霓虹）。

---

## 一、产品是什么（一句话定位）

macOS 本地优先的**中文小说创作 + 分析工具**：上传/输入小说 → 用本机大模型**拆解(逆向工程)、生成正文、连续性检查、仿写**，全部数据存本地 SQLite。用户=网文/小说作者。界面全中文。

---

## 二、页面清单（12 个主屏 + 持久外壳）

### 持久外壳（每个页面都有）
- **顶部全局导航 GlobalNav**：左品牌「Novel Local AI」+ 中部三大入口（项目 / 运行记录 / 设置）+ 右「本地数据·SQLite」徽标 + 运行记录上的待审批红点。当前是唯一深色区(#11171b)。

### A. 项目区
1. **首页 / 项目列表 (Dashboard)** `#/projects`：页头大标题「分析、解构、仿写小说」+ 两个主操作（上传小说分析 / 新建项目）+ 四宫格统计（待审批 / 运行中 / 失败 / 可用模型）+ 三列项目卡片网格 + 空状态。
2. **项目总览 (Overview)** `/overview`：深色英雄引导卡「下一步行动」+ 4 格统计条 + 模型就绪度卡 + 故事设定完整度 + 最近章节 / 最近运行两列。
3. **★章节工作区 (ChaptersV2)** `/chapters`：**核心创作界面，三栏**——左章节列表 / 中正文编辑（大字标题输入+目标+大纲+人物多选+引用搜索）/ 右 Loop 配置与启动面板（运行模式、Provider、高级分模型、当前运行卡、上下文检查器）。
4. **创作中心 (CreativeStudio)** `/creative`：三模式合一（想法生成 / 拆解 / 仿写）。顶部模式切换 + 左输入区（大文本域 + 上传框 + 两级模型选择 + 拆解维度多选 + 进度条）+ 右侧栏 + 底部候选卡列表（逐条接受/拒绝/批量按置信度）。
5. **人物卡 (CharacterCards)** `/characters`：左列表 + 右编辑表单（角色字段 + JSON 文本域）。
6. **世界观设定 (Worldbuilding)** `/world`：左规则列表(带优先级徽章) + 右编辑表单（名称/分类/优先级/描述）。
7. **Loop Runs 列表 (RunListPage)** `/runs`：状态筛选胶囊 + 多章生产线面板 + Loop Run 五列表格行。
8. **★Run 详情 (RunDetailPage)** `/runs/{id}`：**状态最丰富的页**——头部状态徽标 + 失败/暂停/审批等多色横幅 + 左栏实时初稿流式预览/不可变版本预览 + 右栏（自动策略卡 + 状态机时间线 + 连续性报告 + 修订计划折叠）+ 底部 sticky 人工决策栏（批准/拒绝/修订）。

### B. 运行记录区
9. **运行记录 (ActivityListPage)** `#/runs`：聚合所有模型动作（章节/多章/拆解/创作）。两排筛选胶囊（类型 + 状态）+ 分组折叠的记录行 + 彩色状态徽标。

### C. 设置区（三 tab）
10. **设置·本地模型 (ModelSettings)** `#/settings`：推荐与任务分配（三角色卡）+ 模型配置（折叠，左列表右表单）+ 高级参数表 + 本地模型清单（扫描卡网格）+ **两张基准结果表**（连续性检查 / 正文写作，含彩色数值单元格 + 结论徽章）。
11. **设置·提示词 (PromptManager)** `#/settings`：双栏——左按四组列出 ~28 个提示词模板 / 右大号代码编辑器。
12. **设置·应用**：本地数据说明卡。

## 三、当前设计系统（要被替换的）

- **配色 token（仅 4 个 + 1 阴影）**：`ink #171512`（近黑/主按钮）、`paper #f5f1e8`（米白卡片底）、`moss #38564a`（墨绿/强调/hover）、`rust #a85535`（铁锈红/小标题徽标）；阴影 `shadow-panel`(暖色长投影)。语义状态色借 Tailwind 默认 红/琥珀/蓝/绿。
- **字体**：正文 Inter（系统回退，未自托管）；标题/小说正文 `font-serif`（落 Georgia 默认栈，未自定义）；代码 mono。**改版应显式引入 webfont。**
- **复用类（改版要重做的）**：`.panel`(圆角卡片)、`.field`(输入框)、`.label`(全大写小标题)、`.btn-primary/.btn-soft`(按钮)。圆角 rounded-xl/2xl 为主，柔和长投影。
- **当前气质**：暖纸质、编辑部、克制内敛、衬线书卷感。高信息密度、桌面布局。
- **硬编码色散落处**（改主题要同步）：`index.css`(根背景/渐变)、`GlobalNav.tsx`(#11171b/#d79b78)、`index.html`(theme-color)、`tailwind.config.js`(token)。

## 四、目标风格：赛博朋克 / 边缘行者（要具体描述给 Claude Design）

- **底色**：近黑/深蓝黑（#0a0a0f ~ #0d0221）打底，绝非纸白。
- **霓虹主色**：青(cyan #00f0ff)、品红/洋红(magenta #ff003c / #ff2a6d)、电黄(#fcee0a)、紫(#bd00ff)。高饱和、强对比、发光。
- **质感效果**：霓虹辉光(glow/box-shadow)、扫描线(scanlines)、故障风(glitch)、色差(chromatic aberration)、网格/HUD 框线、全息渐变、噪点、单色等宽科技字。
- **字体**：标题用科技/等宽展示字（如 Rajdhani / Orbitron / Chakra Petch / JetBrains Mono 之类），正文可配一款清晰无衬线；中文需挑一款有科技感又能正常显示中文的字（这点要特别提醒 Claude Design——很多炫酷英文字不含中文，中文得单独配）。
- **动效**：霓虹描边流动、hover 辉光、数据流/打字机、状态徽标呼吸闪烁。
- **氛围参考**：Edgerunners 的 HUD/义体界面、2077 的菜单与 braindance 界面、Blade Runner 霓虹街景。
- **保留**：信息密度与三大区结构不变（导航/项目/设置），只换皮+换氛围，别破坏可用性。
