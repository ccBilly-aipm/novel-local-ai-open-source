# Open-source edition

本仓库是 Novel Local AI 的公开源码版本，首个基线为 `v1.0.0`。

公开版包含应用源码、测试、迁移、Prompt、设计文档和示例截图；不包含：

- 用户 SQLite 数据库及 WAL/SHM；
- API Key、`.env` 和本机服务配置；
- 本地模型权重、缓存和下载产物；
- 运行日志和部署备份；
- 用户小说正文或私人项目数据。

公开版与维护者的闭源备份仓库是两个独立 Git 仓库。公开版的变更应通过常规 commit/PR 合入，不应依赖闭源仓库历史。

许可证为 Apache-2.0。第三方依赖、Provider 服务和模型权重继续受各自许可证约束。
