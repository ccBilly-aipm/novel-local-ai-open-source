# 第六章 转移

墙上的电子钟跳到了 21:48。

周明最后检查了一遍口袋。那枚存储着违规记忆的芯片贴在他的肋骨上，隔着衬衫，触感冰凉。他深吸了一口气，将外套拉链拉到底，遮住了右手手背那道旧疤，然后转身走向档案馆大门。

“周明，还没走？”

前台的保安探出头，眼神在周明脸上停留了一瞬。

“加班完了。”周明简短地回答，声音压得很低，没有回头，“今晚有点急事。”

保安点了点头，没再多问。22:00 是市档案馆的死线，一旦过了这个点，铁门与电子锁联动断电，任何人都无法在夜间进入或停留。这是规则 B，也是周明必须遵守的底线。他不能晚到，更不能迟到。

推开厚重的玻璃门，夜风裹挟着初冬的寒意扑面而来。周明没有打伞，径直走向地铁站。

地铁车厢里人很少。周明特意避开了拥挤的早高峰和晚高峰线路，选择了一条需要换乘三次的冷门路线。他始终贴着车门站立，余光不断扫视着身后和车厢连接处的阴影。

在换乘通道里，他停下了脚步。

前方三米，一个穿着灰色连帽衫的男人也在等车。那人低着头看手机，帽檐压得很低。周明握紧了口袋里的芯片，指节微微发白。他向左跨了一步，走进了另一节车厢，随后迅速拉下窗帘，将视线隔绝在窗外飞逝的广告牌上。

他在心里默数。三十秒。一分钟。

直到列车广播响起“下一站，西郊站”，周明才感觉到后背的冷汗稍微退去。

西郊是废弃信号塔的所在地。这里早已没有信号，只有锈蚀的铁架矗立在荒草中，像某种巨大的墓碑。

林秋已经到了。她穿着一件黑色的冲锋衣，站在塔架阴影里，手里提着一个旧工具箱。

“来了？”林秋的声音很轻，像是在怕惊动什么。

“嗯。”周明走到她面前，两人没有拥抱，也没有寒暄。

他从口袋深处掏出芯片，递了过去。金属外壳在月光下泛着冷光，上面没有任何标识，这是陈砚留下的最后一样东西。

“接稳了。”周明低声说。

林秋伸手接过，动作熟练地将芯片塞进工具箱侧面的夹层里，随即扣好锁扣。

就在这时，草丛里传来一阵窸窣声。

两人同时僵住。周明猛地侧身，右手下意识地摸向腰间，却发现那里空荡荡的——他早已习惯了在档案馆使用删除仪，那里本该有工具，但此刻什么都没有。

脚步声从铁架下方传来，沉重，且杂乱。

“别动。”林秋示意周明噤声。

一个摇摇晃晃的身影从草丛里钻了出来。是个醉汉，手里提着一瓶啤酒，嘴里嘟囔着听不懂的歌词。他踉跄着走过塔架旁，完全没有注意到阴影里藏着的两个人。

周明屏住呼吸，直到那瓶啤酒落地碎裂的声音消失在远处，才缓缓吐出一口气。

“虚惊一场。”林秋拍了拍胸口，“最近风声紧，我看了，这附近多了几个生面孔。”

“不管是谁，只要没看到交接过程就好。”周明摸了摸口袋，确认芯片已经不在，“接下来你打算藏在哪？”

“老地方，你不用管。”林秋合上工具箱，眼神里闪过一丝复杂的情绪，“周明，这件事得快点定夺。陈砚的笔记烧了，但芯片还在，这东西就像定时炸弹。”

“我知道。”周明点了点头，语气平淡，“只要不被人发现，它就是安全的。”

“希望如此。”

两人对视一眼，随即背道而驰。林秋向南，周明向北。

回程的路上，周明坐了一辆出租车。车窗映出他疲惫的脸，左眼那抹淡蓝色在霓虹灯下显得愈发幽深。他靠在椅背上，试图平复心跳，但那只握着手机的手却始终没有放松。

手机屏幕突然亮起，震动了一下。

是一条来自苏婷的消息。

周明的手指顿住了。

“周明，今晚 22:00 前的监控有个死角，你在那停留了五分钟，正常下班不需要这么久吧？”

发信时间：21:55。

那是他离开档案馆前的时间。也就是说，在他刚走出大门没多久，苏婷就已经查了监控，并且注意到了他在监控死角停留的异常。

周明盯着那行字，指尖微微发凉。苏婷是新人实习生，按权限她不应该能调取监控死角的数据，除非……她做了手脚，或者有人帮她。

“怎么了？”出租车司机透过后视镜看了一眼，“到了，前面路口。”

“到了。”周明收回手机，将屏幕扣在掌心。

他推开车门，冷风灌进领口。街道空旷，路灯昏黄。

在走进公寓楼的阴影前，他再次点亮屏幕，删掉了那条消息，却没有回复。

苏婷已经起疑了。而比怀疑更危险的，是那种被窥视的感觉，正像潮水一样，顺着这条看不见的线，慢慢漫过他的脚踝。
