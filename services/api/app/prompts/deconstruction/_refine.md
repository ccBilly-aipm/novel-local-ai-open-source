## 可迁移层级 layer（三选一）
- surface：表层具体物——具体人名、地名、独有的具体情节。仿写时**必须换皮，禁止照搬**。
- pattern：可复用模式——结构骨架、节拍套路、爽点公式、人物关系范式、伏笔回收方式。仿写时**可照搬骨架**。
- signature：灵魂指纹——文风、叙事腔调、母题、世界规则的"类型"与质感。仿写时**要复刻其神**。

## 仿写可复用度 reuse_score（0~10 的数字）
综合三点打分：证据强度 + 具体可操作性 + 对文风/结构的承载度。越是"照着这条就能直接写出来"，分越高。

## 硬规则
- 必须对清单里的每个 ref 给出标注；ref 只能用清单给出的编号。
- reuse_note 用一句话中文写"仿写时具体怎么用这条"。
- 不臆造原文没有的事实，只做归类与可操作化，不改写原候选内容。
